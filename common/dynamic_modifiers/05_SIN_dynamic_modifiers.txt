# There are some things you have to edit or add to create a new idea for a country
#
# Step 1: Make Your own Dynamic Modifier
# \Steam\steamapps\workshop\content\394360\2777392649\common\dynamic_modifiers\05_SIN_dynamic_modifiers.txt
#
# Modifier data base:
# \Steam\steamapps\workshop\content\394360\2777392649\localisation\english\replace\replaced_from_focus_l_english.yml
# \Steam\steamapps\workshop\content\394360\2777392649\localisation\english\replace\replaced_from_vanilla_modifiers_l_english.yml
#
# Step 2: Add Dynamic Modifier Into Your Country Input
# \Steam\steamapps\workshop\content\394360\2777392649\history\countries\SIN - Singapore.txt
# 	530 = { add_dynamic_modifier = { modifier = SIN_singapore_problem } }
#
# Step 3: Registered Your GFX
# \Steam\steamapps\workshop\content\394360\2777392649\interface\MD_ideas.gfx
# \Steam\steamapps\workshop\content\394360\2777392649\interface\MD_dynamic_modifiers.gfx
#
# 	name = "GFX_idea_SIN_singapore_problem"
#	texturefile = "gfx/interface/state_modifiers/SIN_singapore_problem.dds"
#
# Step 4: Registered your State Modifier into selected state
# \Steam\steamapps\workshop\content\394360\2777392649\history\states\
#
# Step 5: Create Idea Icon
# \Steam\steamapps\workshop\content\394360\2777392649\gfx\interface\state_modifiers
# 1. In creating icons on idea, Millenium Dawn has criteria or standards to standardize the visual style, which we can discuss on the Discord Channel "graphics-sound"
# 2. To get the template, you can download it from Google Drive which is provided in the pin Discord channel "graphics-sound" section.
# 3. Create an icon that represents the idea you have.
# 4. To ensure the visual style is not weird and uniform, you have to go through the bureaucracy of checking the icon by the GFX reviewer on the Discord Channel gfx_request, if it meets the requirements, the GFX reviewer will notify you.
#
# Step 6: Add Localisation
# \Steam\steamapps\workshop\content\394360\2777392649\localisation\english\MD_focus_SIN_l_english.yml
# Add your dynamic modifier into localisation, adding name and description
